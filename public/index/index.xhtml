Progressive Web Application
PWAs are handy "apps" you can install from a browser. They only require a compatible browser to work, which means they can be "installed" on almost any platform. {app.name} has an installable PWA, so you can easily access it from wherever in your environment, on many platforms, with just one click or tap away.
Privacy, 256-Bit AES Encryption
{app.name} is focused around privacy. All the data you create, be it to-dos, notes, your calendar or contacts, is stored encrypted, so nobody but you can access it. Even the keys are stored encrypted, and are only decrypted locally in your client using a password derived key. This means your password is never sent to the server in the first place.
Open Source
Another important aspect worth denotation is {app.name} being open source. It provides sufficient documentation, so you can host it yourself too. Or you can just take a look at its source code and see how everything works for yourself.
